Detailed Description

Classe que representa um raio em um espaço tridimensional.

A classe ray é utilizada para representar um raio que possui uma origem (point3) e uma direção (vec3) no espaço tridimensional. Oferece métodos para acessar a origem, a direção e calcular um ponto em determinada distância ao longo do raio.
