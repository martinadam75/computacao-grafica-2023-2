\doxysubsection{Detailed Description}
Classe que representa um intervalo na reta real. 

A classe {\ttfamily interval} é utilizada para representar intervalos fechados na reta real, oferecendo métodos para operações comuns como expansão, verificação de pertencimento, entre outras.
